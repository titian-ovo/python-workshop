🐍 Python 安装教程

新手必看！5 分钟完成 Python 环境配置

1 下载 Python 安装包

访问 Python 官网下载最新版本：

📥 前往 Python 官网下载
💡 提示：官网会自动识别你的操作系统，推荐下载最新的 Python 3.x 版本（目前最新是 3.12+）

🪟 Windows 系统

  1. 点击下载 "Python 3.12.x Windows installer (64-bit)"
  2. 等待下载完成（约 20-30MB）
  3. 双击下载的安装文件（如 python-3.12.5-amd64.exe）

🍎 macOS 系统

  1. 点击下载 "macOS 64-bit universal2 installer"
  2. 等待下载完成（约 30MB）
  3. 双击下载的 .pkg 文件

🐧 Linux 系统

大多数 Linux 系统已预装 Python，使用包管理器安装：

# Ubuntu/Debian sudo apt update sudo apt install python3 python3-pip # CentOS/RHEL sudo yum install python3 python3-pip # Arch Linux sudo pacman -S python python-pip
2 安装 Python（Windows 重点！）
⚠️ 重要：Windows 用户安装时必须勾选 "Add Python to PATH"！

Windows 安装步骤：

  1. 运行安装程序后，第一步就会看到 "Add python.exe to PATH" 选项
  2. ✅ 务必勾选这个选项！（这是最关键的一步）
  3. 点击 "Install Now"
  4. 等待安装完成（约 2-3 分钟）
  5. 看到 "Setup was successful" 点击 "Close"
💡 为什么一定要勾选 "Add Python to PATH"？
这样你可以在任何地方（命令提示符、PowerShell、VS Code 等）直接使用 python 命令，不需要输入完整路径。

macOS 安装步骤：

  1. 双击 .pkg 文件
  2. 点击 "继续" → "继续"
  3. 同意许可协议
  4. 选择安装位置（默认即可）
  5. 点击 "安装"，输入密码确认
  6. 等待安装完成
3 验证安装是否成功

Windows 验证：

  1. 按 Win + R 打开运行窗口
  2. 输入 cmd 回车，打开命令提示符
  3. 输入以下命令：
python --version

如果显示 Python 3.12.x（或类似版本号），说明安装成功！✅

macOS/Linux 验证：

  1. 打开 "终端"（macOS）或终端（Linux）
  2. 输入：
python3 --version

显示版本号即成功！✅

💡 测试运行 Python：
# 输入 python (Windows) 或 python3 (Mac/Linux) 进入交互模式 >>> print("Hello, Python!") Hello, Python! >>> exit() # 退出
4 安装代码编辑器（推荐 VS Code）

写代码需要一个好用的编辑器，强烈推荐 Visual Studio Code（免费、强大、易用）

📥 下载 VS Code

VS Code 安装步骤：

  1. 访问官网下载安装包
  2. 运行安装程序，一路 "下一步" 即可
  3. 安装完成后打开 VS Code
  4. 安装 Python 扩展：
    • 点击左侧 "扩展" 图标（或按 Ctrl+Shift+X）
    • 搜索 "Python"
    • 找到 Microsoft 出品的 "Python" 扩展
    • 点击 "安装"
💡 其他编辑器选择：
  • PyCharm Community - 功能强大的专业 Python IDE（免费）
  • Sublime Text - 轻量快速
  • Notepad++ - 简单够用
5 运行你的第一个 Python 程序

方法 1：交互模式（最简单）

  1. 打开命令提示符（Windows）或终端（Mac/Linux）
  2. 输入 python（Mac/Linux 用 python3）回车
  3. 看到 >>> 提示符后，输入：
>>> print("Hello, World!") Hello, World! >>> exit()

方法 2：运行脚本文件

  1. 在桌面新建一个文本文件，命名为 hello.py
  2. 用记事本或 VS Code 打开，输入：
print("Hello, World!") print("我的第一个 Python 程序！")
  1. 保存文件
  2. 打开命令提示符，进入桌面：
cd Desktop
  1. 运行程序：
python hello.py

看到输出就成功了！🎉

❓ 常见问题解答

Q1: 输入 python 提示 "不是内部或外部命令"？

原因：安装时没有勾选 "Add Python to PATH"

解决方法：
  1. 重新运行 Python 安装程序
  2. 选择 "Modify"
  3. 确保勾选 "Add Python to environment variables"
  4. 完成安装

Q2: Windows 商店的 Python 能用吗？

可以用，但不推荐。建议从官网下载，版本更新、功能更全。

Q3: 应该学 Python 2 还是 Python 3？

绝对学 Python 3！ Python 2 已在 2020 年停止支持，所有新项目都用 Python 3。

Q4: 需要安装 Anaconda 吗？

初学者不需要。Anaconda 适合数据科学，但体积大（3GB+）。先用官方 Python，以后需要再装。

Q5: 安装后找不到图标？

Python 没有桌面图标，它是命令行工具。用 VS Code 写代码，或在命令行输入 python 运行。
